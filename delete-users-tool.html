🗑️ Delete All Non-Admin Users

Remove all registered users except admin accounts

⚠️ WARNING: This will permanently delete all non-admin users! This action CANNOT be undone!
💡 What this does:
✅ Keeps all admin users safe
❌ Deletes all regular/unverified users
🔄 Allows you to register fresh test accounts

Step 1: Open Supabase Dashboard

Go to: Supabase Dashboard

Step 2: Go to SQL Editor

Click on "SQL Editor" in the left sidebar

Step 3: Copy & Run This SQL

-- First, see which users will be deleted SELECT id, email, email_confirmed_at, created_at, '❌ WILL BE DELETED' as status FROM auth.users WHERE id NOT IN ( SELECT user_id FROM user_roles WHERE role = 'admin' ) ORDER BY created_at DESC; -- Delete all non-admin users DELETE FROM auth.users WHERE id NOT IN ( SELECT user_id FROM user_roles WHERE role = 'admin' ); -- Verify only admins remain SELECT id, email, email_confirmed_at, '✅ REMAINING (Admin)' as status FROM auth.users ORDER BY created_at DESC;

Step 4: Click "Run"

Paste the SQL and click the "Run" button or press Ctrl+Enter

✅ Done! All non-admin users have been deleted. Now you can register fresh accounts!
💡 After deletion:
• Go to /auth page
• Register with a NEW email address
• Verify your email
• Login and test the application!
