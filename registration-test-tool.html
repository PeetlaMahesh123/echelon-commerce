📱 Registration & Verification Guide

Complete setup for desktop and mobile testing

💻 Desktop Registration Flow
Test on your computer first, then move to mobile

1Open Registration Page

Go to: http://localhost:8081/echelon-commerce/auth

2Create Account

Click "Sign Up" tab and enter:
• Full Name (any name)
• Real Email Address (Gmail/Outlook)
• Password (min 6 characters)

3Verify Email

1. Check your email inbox
2. Find email from Supabase/Echelon Commerce
3. Click "Confirm your email" link
4. Browser will open and verify your account

4Login

Return to /auth page, click "Sign In", enter credentials, and login!

Open Registration Page
⚠️ Important: Complete Supabase configuration first!

1Find Your Computer's IP

Open Command Prompt and run:

ipconfig | findstr IPv4

Note the IPv4 address (e.g., 192.168.1.9)

2Configure Supabase

Go to Supabase Dashboard > Authentication > URL Configuration
Add these redirect URLs:
http://192.168.1.*
http://192.168.1.9

3Access from Mobile

On your phone browser, go to:
http://YOUR-IP:8081/echelon-commerce/auth

4Register & Verify

Same as desktop - register with real email, check email on phone, click verification link!

✅ Mobile Testing Checklist:
  • Phone and computer on same WiFi
  • Using computer's IP address (not localhost)
  • Supabase redirect URLs configured
  • Using real email you can access on phone
👑 Two Ways to Become Admin

1Use Admin Tab

Go to /auth and click the "Admin" tab (not Sign Up)

2Create Admin Account

Enter your details and click "Create Admin Account"

3Verify & Login

Check email, verify, then login - you'll have admin access automatically!

1Register as Regular User

Sign up normally using "Sign Up" tab

2Login

Sign in with your credentials

3Click "Make Me Admin"

On the login page, there's a "Make Me Admin" button - click it!

4Access Admin Panel

You'll be redirected to admin dashboard with full access!
